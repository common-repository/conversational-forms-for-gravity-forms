Convert Gravity forms to Conversational design easily and without coding.

== Description ==

Conversational Forms for Gravity Forms lets you easily convert Gravity Forms to conversational design. There are built in themes and layout options.

Here are some of the features of Conversational Forms for Gravity Forms

* Layouts: There are two layouts options right now. A Default layout and a [premium PRO layout](https://gravityconversational.com/downloads/gravity-conversational-pro/).

* Themes: There are built in themes which you can apply. By using premium conversational styler addon you fully customize the colors of themes with the help of color picker or use hex codes.

* Font Family: You can change font family for labels and inputs.

* Font Size: You can choose from a set of font sizes. There is setting for responsive font size using which you can assign different font sizes for different devices ( desktops, tablets and mobiles ).

* Spacing : Spacing controls includes padding for inputs and buttons. These are also responsive and can have different values according to devices.

* Animations: You can also aadd animations to inputs and buttons.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/wpm-gravity-conversational-core` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
